
# 温馨提醒特效程序集

一个包含多种视觉特效的Python温馨提醒工具包，基于tkinter和pygame开发，提供丰富的动态展示效果。

## 一、程序集合

### 1. Happy.py - 多窗口温馨提醒
- **版本**：5.0
- **功能**：多窗口分批弹出温馨话语
- **特性**：
  - 窗口数量可自定义（默认上限400，可修改`max_windows`参数调整）
  - 随机背景色与内容
  - 自动/手动关闭选项
  - 空格键快速关闭全部
- **运行**：`python Happy.py`

### 2. Happy_Love.py - 爱心爆炸特效
- **版本**：简化版
- **功能**：全屏爱心动画与文字爆炸效果
- **特性**：
  - 爱心粒子数量可通过参数调整
  - 多彩文字配色
  - 全屏无框显示
  - 文字留存时间可自定义（修改`AUTO_CLOSE_TIME`参数）
- **运行**：`python Happy_Love.py`

### 3. happy_Down.py - 文字雨程序
- **版本**：9.1 短语文字雨版
- **功能**：温馨短语从顶部下落形成文字雨
- **特性**：
  - 文字数量可调整（默认上限200，修改`max_windows`参数）
  - 拖尾与旋转动画
  - 点击交互功能
  - 背景音乐支持
  - 多彩文字效果
- **运行**：`python Happy_Down.py`

### 4. happy_Plus.py - 纯文字特效
- **版本**：8.0 纯文字震撼版
- **功能**：无边框文字动画展示
- **特性**：
  - 文字数量可自定义（默认上限200，修改`max_windows`参数）
  - 多种动画效果（打字机/彩虹/爆炸等）
  - 文字粒子系统
  - 鼠标悬停交互
  - 音效与背景音乐
- **运行**：`python Happy_Plus.py`

### 5.Happy_Up
- **功能**：短语从底部上升并爆炸成多个子短语
- **特性**：
  - 批次数量与每批短语数可自定义（`batch_count`/`batch_size`参数）
  - 爆炸粒子数量可调整（默认15-25个，修改`explosion_count`参数）
  - 物理引擎模拟（重力/减速）
  - 透明度渐变效果
  - 随机轨迹与速度

## 二、通用操作

| 按键 | 功能 |
|------|------|
| 空格键 | 关闭所有窗口并停止生成 |
| ESC键 | 强制退出程序 |
| 鼠标点击 | 关闭单个窗口（部分程序） |

## 四、使用指南

### 环境要求
- Python 3.6+
- 依赖库：`pip install pygame`
- tkinter（通常随Python预装）


### 故障排除
- **窗口不显示**：检查tkinter安装，确认屏幕分辨率适配
- **程序卡顿**：减少`max_windows`值，增大`CREATE_DELAY`间隔
- **音乐问题**：检查pygame安装完整性，确认音频文件路径正确
